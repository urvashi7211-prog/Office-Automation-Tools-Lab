# 🖥️ Office Automation Tools Lab Programs

This repository contains solutions to various **Office Automation Tools Lab Assignments** completed as part of my academic coursework.

## 📚 About

The purpose of this repository is to practice and demonstrate skills in commonly used Microsoft Office applications:

- 📊 MS Excel
- 📝 MS Word
- 📽️ MS PowerPoint

Each lab task is organized and documented for learning, reference, and academic purposes.

## 🎯 Objectives

- Learn the fundamentals of Office Automation Tools.
- Create professional documents using MS Word.
- Perform calculations, formulas, and data analysis using MS Excel.
- Design effective presentations using MS PowerPoint.
- Develop practical productivity skills used in academic and professional environments.
